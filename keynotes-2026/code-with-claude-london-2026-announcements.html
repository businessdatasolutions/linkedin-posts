Opus 4.7 powers Claude Design (Anthropic Labs). Already shipped — currently the frontier model Global (via Claude API and products) Claude API, Claude Code, Claude.ai

Used by the keynote as the forward-looking signal that future Claude models will be "more capable than Opus 4.7 and even Mythos preview". Preview (no GA date stated) Global (preview cohort) Preview access (gating not specified)

Claude Platform — Claude Managed Agents (Angela & Caitlyn)
5 Advisor strategy (recently shipped) API pattern that splits execution from advising. A smaller model (Haiku or Sonnet class) executes; when it needs help it reaches out to a larger model (Opus) for advice. Implemented by updating the tools array on the Messages API — no rebuild required. EvenUp Legal reported "frontier model quality at 5× lower cost". Best for premium product paths and high-volume agentic workloads where ROI matters. Available now Global Claude API (Messages API)
6 Claude Managed Agents (recap, with new features below) Agentic harness paired with production-grade infrastructure. Reported to let teams ship "10× faster" and reach production-grade agents in days, not months. Anchor customer: Asana built AI Teammates on Managed Agents so humans collaborate with agents directly inside Asana projects. Already launched Global Claude Developer Platform
7 Multi-agent orchestration / Outcomes / Dreaming (recently shipped) Three Managed Agents upgrades shipped recently: Multi-agent orchestration (build fleets of agents); Outcomes (specify what success looks like, Claude iterates to get there); Dreaming (Claude introspects its own previous transcripts to learn and self-improve). Available now Global Claude Managed Agents
8 Self-hosted sandboxes — new today Claude Managed Agents now execute work on your server, not just Anthropic's. When the agent needs to run code or edit files, it puts a work item on a queue that your provider picks up and spins up a sandbox in your own account. First-class integration support for Daytona, Cloudflare, Vercel and Modal sandboxes. Available today Global Claude Managed Agents (+ your sandbox provider account)
9 MCP tunnels — new today Lets Managed Agents reach MCP servers that live behind your firewall on your private network — no public-internet exposure. You set up a gateway on your private network and establish a secure connection to Anthropic; agents call MCP servers via tunnel.anthropic.com URLs. Configured directly in the Claude Developer Console. Available today Global Claude Managed Agents + Claude Developer Console
Claude Code (Cat & Boris Cherny)
10 Claude Code Desktop app — new updates today Full-screen graphical interface for Claude Code. Built-in previews, sidebar control plane, image and rich-output rendering. Single unified view across local sessions and cloud sessions with visual indicators of which agents are running, blocked, or need input. Built on the Claude Agent SDK. Latest updates available today Global Claude Code subscription
11 Claude Agents view (CLI) — new New surface for Claude Code users who prefer the terminal. Single-pane view of what's running, what's waiting on you, and what's done. Inline reply to unblock, and jump in and out of sessions without losing your place. Available today Global Claude Code subscription
12 Code Review product (recap) Deploys a team of agents to traverse all code changes and auxiliary files looking for critical bugs. "Thousands of companies use this every day, including every internal Anthropic team." Already shipped Global Claude Code
13 Claude Code on iOS & Android (Remote Control) (recap) Fire off Claude Code tasks from your phone — leave the laptop closed, walk outside, still ship work. Already shipped Global (mobile) Claude Code
14 Autofix (PRs) (recap) Listens for flaky CI, code-review comments and merge conflicts on your open PR and proactively fixes them so the PR stays green without manual babysitting. Already shipped Global Claude Code
15 Routines — new Configure once and Claude Code runs on a schedule, in response to a webhook, or on an arbitrary API call. Demoed in the keynote as the way a new GitHub issue gets a working PR overnight without anyone prompting. Boris frames Routines as a "higher-order prompt" — you write the routine that prompts Claude, instead of prompting Claude yourself. Runs locally or on remote cloud compute. Available today Global Claude Code subscription
16 CI autofix (in Claude Code Desktop) — new Watches a PR that a session just opened and "babysits the PR to production" — autofixes code review and security review comments, autofixes CI failures, auto-rebases on merge conflicts. Demo example: a network-timeout CI failure that the routine diagnosed as a known infra issue and retried. In the Claude Code codebase, Anthropic configures it to fix the root cause instead of just retrying. Available today Global Claude Code Desktop
17 Claude Security (recap) Overnight codebase scan that flags vulnerabilities ranked by severity and lets you kick off Claude Code to address each. Addresses the bottleneck where security teams can't keep up with the volume of code landing. Already shipped Global Claude Code / Claude Platform
18 VS Code IDE extension (recap) IDE extension giving the same agent as the CLI but with visual follow-along of code changes as the agent works. Built on the Claude Agent SDK — the same SDK external developers build on. Already shipped Global Claude Code
