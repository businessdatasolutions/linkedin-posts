Google I/O '26 — Developer Keynote: product & feature announcements

Source: Developer Keynote (Google I/O '26), Google for Developers, 2026-05-19 (58:47). Compiled 2026-05-20. Region column reflects what the keynote actually stated — most launches are global developer products with no geographic carve-out called out on stage.

# Announcement Short description Availability Regions Subscription / access
Chapter 2 — Building Agents (Gemini API & AI Studio)
1 Managed Agents in Gemini API Single Gemini API call returns an agent and a Google-hosted remote Linux sandbox for secure, isolated execution. Custom instructions, tools and data attach through that one call; Google handles all infrastructure. Early access customers include Ramp, Resemble AI and Klipy. Available today (GA) Wherever the Gemini API is available (not specified in keynote) Gemini API (paid usage)
2 Custom agent templates in AI Studio Open-source agent templates pre-loaded with skills and tools defined as markdown — e.g. AI Talk Radio Agent, Customer Support Agent. Source files are editable directly in the AI Studio playground. Available today Wherever AI Studio is available Google AI Studio (free tier)
3 AI Studio → Cloud Run instant deploy (no credit card for new users) Brand-new users can deploy an AI Studio app to a live URL on Cloud Run in a few clicks, without a credit card. Existing Google Cloud projects are also supported. Starting today Cloud Run regions (not specified in keynote) Google AI Studio + Google Cloud (free entry path)
4 Google Workspace integration in AI Studio Prompt-driven connections to Docs, Gmail and Calendar from inside AI Studio Build, alongside the existing Firebase, Firestore, Multichat, Web Search and Nano Banana image generation integrations. Officially announced today Wherever AI Studio + Workspace are available Google AI Studio + Workspace account
5 Build Android apps in AI Studio Go from idea to Kotlin-based Android app entirely in AI Studio — no SDKs, no local environment. Built-in Android Emulator preview; plug in a physical phone to install. Starting today Wherever AI Studio is available Google AI Studio
6 Google Play Store publishing from AI Studio Connect a Play Developer Account and push the app straight to the internal test track from AI Studio's new Publish flow. Testing on your own device works today; sharing with trusted testers rolls out later this summer. Own-device testing today; trusted-tester sharing later summer 2026 Wherever Google Play developer accounts are accepted AI Studio + Google Play Developer Account
7 AI Studio mobile app Brand-new mobile companion that brings the AI Studio build experience to your phone. Rolls out "in a few weeks"; pre-register today Not specified in keynote Google AI Studio
8 Antigravity SDK Programmatic access to the same Gemini-optimised agent harness used by Antigravity, deployable on your own infrastructure for full customisation and control. Launching today Global (developer SDK) Free SDK; Gemini API usage charges apply
9 One-click export from AI Studio to Antigravity Ports the full file system and complete context from an AI Studio project into Antigravity so you can continue in a local development environment without rebuilding state. Available today Global AI Studio + Antigravity (both free entry points)
Chapter 3 — Antigravity
10 Google Antigravity 2.0 (desktop app) New "agent-first" desktop application. Run multiple agents across multiple projects and worktrees in parallel — one vibe-codes the marketing site, another generates brand assets, a third plans architecture, with no collisions. Usable alongside any IDE. Available today (announced earlier in the main keynote) Global Free developer tool; uses Gemini API quota
11 Dynamic Subagents Agents can spawn specialised helpers (e.g. QA subagent, Data Science subagent) that work in parallel on large tasks instead of overwhelming a single agent. Available today Global Antigravity 2.0
12 Scheduled Tasks Standard cron scheduling for agents — e.g. summarise pending PRs every morning, monitor cloud health every hour. Puts agents on autopilot for proactive workflows. Available today Global Antigravity 2.0
13 Antigravity CLI Terminal-native counterpart to Antigravity 2.0 — same harness, same models, same agents, with CLI-tailored UX (themes, keybindings). Includes new /btw command to fork a conversation. Audio input is supported via the latest audio-understanding model. Available today, rolled out to all Gemini CLI users; migration guides published Global Free; Gemini API usage applies
14 One-click setup for Android, Firebase & web in Antigravity Pre-wired integrations for building in the Google ecosystem with zero configuration. Stack remains otherwise agnostic — no vendor lock-in. Available today Global Antigravity
15 Domain-specific skills bundles — Science bundle First in a planned series of domain skill bundles. Equips agents with primitives for health, biology and scientific research workflows. Starting today Global Antigravity
16 Antigravity for Enterprise (Google Cloud connection) Antigravity can now connect directly to enterprise Google Cloud projects under standard enterprise terms. Available today Wherever Google Cloud is available Google Cloud enterprise customers
17 Antigravity in Gemini Enterprise Antigravity will roll out to existing Gemini Enterprise customers. Coming months Wherever Gemini Enterprise is sold Gemini Enterprise
Chapter 4 — Android
18 Official Android support in Antigravity First-class Android development inside Antigravity, complementing native AI Studio Android support. Available today Global Antigravity
19 Android CLI (built into Antigravity, now stable) Toolset for agents covering the full Android workflow without Android Studio: SDK download, project creation, running on devices. Internal testing showed ~70% fewer tokens and up to 3× faster task completion. Stable today Global Antigravity
20 Android Knowledge Base Specialised data source that lets agents search and fetch the latest official Android developer guidance — including coverage of new form factors like Display Glasses that most LLMs don't know yet. Available today (part of Android CLI) Global Android CLI / Antigravity
21 Open-sourced Android Skills Curated skill set encoding best practices — edge-to-edge support, migrating from XML to Compose, Jetpack Navigation 3 migrations, R8 analysis, App Links setup, etc. Available today Global Free / open source
22 Gemini Nano 4 (on-device model) Latest on-device Gemini Nano generation, used in the demo for offline AI trip-summary on a Samsung Galaxy S26 Ultra. Apps can use hybrid Firebase logic to fall back to cloud only when an on-device model isn't available. Referenced as latest (no explicit GA date in this keynote) On supported Android devices Free on-device
23 Samsung Galaxy S26 Ultra on Android Device Streaming The S26 Ultra will be addable to the Android Device Streaming pool so developers can deploy and test on a real device they don't own. Later this summer 2026 Wherever Android Device Streaming is offered Android Studio / Google Play Console
24 R8 configuration Analyzer in Android Studio Analyses keep-rules and build configuration, recommends switching to R8 full mode, and reports optimisation, obfuscation and shrinking scores. Drives fewer ANRs, smaller app size and quicker startup. Available in latest Android Studio Global Android Studio (free)
25 App Links Assistant (updated) New URL Mapping Editor walks you through implementing deep links — pick the host, the activity that should handle the intent, and provide a sample URL. AI generates the routing logic. Available in latest Android Studio Global Android Studio (free)
26 Publish to Google Play from Android Studio The "Generate signed app bundle" flow has a new option to upload directly to Play — pushing to the internal test track without leaving Android Studio. Available today Wherever Google Play developer accounts are accepted Android Studio + Play Developer Account
27 Migration Assistant in Android Studio Early sneak peek of a new wizard that migrates apps to Android from iOS, React Native or web. Choose AI-led or Guided Migration; supports Journeys-based agent validation of the resulting app. Also working on Kotlin Multiplatform migration to share business logic with iOS. Coming later in 2026 (sneak peek) Global Android Studio (free)
Chapter 5 — Chrome & Web
28 Modern Web Guidance Expert-vetted, Baseline-aware skill bundle that teaches AI coding agents to use current web-platform features (with fallbacks for the newest ones). Reported +37 percentage-point pass-rate uplift on guided vs unguided web tasks. Over 100 use cases at launch; framework-agnostic. Early preview today Global Free; one-click install in Antigravity; installable as a skill package in other coding agents
29 WebMCP API Proposed browser standard that lets a site expose JavaScript-defined tools to browser-based agents (e.g. an updateCarConfiguration tool exposes the configurator schema). Makes agent interactions precise and reliable instead of relying on DOM scraping. Enters Origin Trial in Chrome 149 Anywhere Chrome 149+ is available (Origin Trial sites) Free (web standard / browser feature)
30 Gemini in Chrome — WebMCP support Shown as a prototype: Gemini in Chrome consumes a site's WebMCP tools, builds an autobrowse plan, asks for confirmation, then executes (demo: "configure the ultimate party car under $40k"). "Soon" (active experiments with ecosystem partners) Wherever Gemini in Chrome is rolled out Gemini in Chrome
31 Chrome DevTools for Agents Closed feedback loop for coding agents made of an MCP server, a CLI and tailored skills. Lets agents observe runtime behaviour, run Lighthouse, read reports and re-audit after fixes — without a human acting as clipboard between DevTools and chat. Available today Global Free; works with Antigravity and 20+ other coding agents
32 Lighthouse "agentic browsing" audit category New Lighthouse category that validates WebMCP tool registrations, declarative form metadata, the new llms.txt file, and re-weights accessibility audits (since agents navigate via the accessibility tree). Available today (part of Chrome DevTools for Agents) Global Free
33 HTML-in-Canvas API Lets you render real DOM elements inside a <canvas> environment so canvas content becomes searchable, accessible, selectable, translatable, autofill-compatible and CSS-styleable — without giving up WebGL/3D visuals. In Origin Trial now Anywhere participating Chrome versions are available Free (web platform feature)
